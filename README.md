## Capgemini Cafe X
An app that takes menu items (Cola, Coffee, Cheese Sandwich, Steak Sandwich) and calculates the bill

## Requirements
Scala 2.12  
ScalaTest 3.0.1  
SBT 0.13.15

## Building the app
`sbt clean package`

## Running the app
`sbt 'run "Cola" "Coffee" "Cheese Sandwich"'`  
or run from your IDE
